# Agent Working Rules (Local)

이 저장소에서 작업을 시작할 때 아래 순서를 반드시 먼저 수행한다.

1. `docs/SESSION_LOG.md`를 먼저 읽고 현재 상태를 파악한다.
2. 로그의 `Open Issues`와 `Next Start Checklist`를 우선 확인한다.
3. 작업이 끝나면 `docs/SESSION_LOG.md`에 새 항목을 추가한다.

기록 규칙:

- 날짜/시간(Asia/Seoul)을 명시한다.
- 사용자 요청, 수행한 변경, 결과, 남은 이슈를 짧고 명확하게 쓴다.
- 실제 파일 경로와 커밋 해시를 남긴다.
