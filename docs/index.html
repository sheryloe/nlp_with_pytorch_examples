Architecture

서비스 구조와 저장소 구조

Service
  • Frontend

    정적 웹 앱을 Vercel에 배포하고 브라우저에서 UI를 처리합니다.

  • Auth / Database

    Supabase Auth, Postgres, RLS로 접근 제어와 사용자별 데이터 분리를 담당합니다.

  • Automation

    Edge Functions가 회원가입, 비밀번호 재설정, 가격 갱신을 처리합니다.

Repository
`web/`

`docs/`

`scripts/`

FAQ

자주 묻는 질문

Donggri Ledger는 어떤 구조로 운영되나요?

프론트엔드는 Vercel, 인증과 데이터베이스는 Supabase, 코드와 문서는 GitHub 기반으로 운영합니다.

현재 어떤 기능을 사용할 수 있나요?

회원가입, 로그인, 카드 결제예정, 자산 관리, 고정지출, 예산, 투자 포트폴리오, 백업과 복원을 사용할 수 있습니다.

왜 카드 결제예정 기능이 중요한가요?

카드를 쓴 날과 실제 돈이 빠지는 날이 다르기 때문에 생활비 흐름을 맞추려면 결제예정 금액을 따로 관리해야 합니다.
