카테고리별 예산과 월 고정지출을 동시에 관리하여 과소비를 방지합니다.

Service Stack

Frontend (Vercel)

정적 웹 앱을 Vercel에 배포하고 브라우저에서 모든 UI와 인터랙션을 처리합니다.

Automation (Edge Functions)

Edge Functions가 회원가입, 비밀번호 재설정, 투자 자산의 가격 실시간 갱신을 처리합니다.

FAQ

자주 묻는 질문

Donggri Ledger는 어떤 구조로 운영되나요?

프론트엔드는 Vercel, 인증과 데이터베이스는 Supabase, 코드와 문서는 GitHub 기반으로 운영하며 서버리스 구조를 최대한 활용합니다.

현재 어떤 기능을 사용할 수 있나요?

회원가입, 로그인, 카드 결제예정 관리, 자산 관리, 고정지출, 예산 세팅, 실시간 가격 기반 투자 포트폴리오, JSON 데이터 백업과 복원을 지원합니다.

왜 카드 결제예정 기능이 중요한가요?

"카드를 쓴 날과 실제 돈이 빠지는 날이 다르기 때문입니다."

생활비 흐름을 정확히 맞추려면 결제예정 금액을 별도로 추적하고 결제 통장의 잔액과 매칭하는 과정이 반드시 필요합니다.
