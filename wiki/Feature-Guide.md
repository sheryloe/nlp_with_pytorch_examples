# Feature Guide

## Account and Access

현재 인증 구조는 아이디 + 비밀번호 기반입니다.

지원 기능:

- 회원가입
- 로그인
- 로그아웃
- 보안질문 3개 저장
- 보안질문 기반 비밀번호 재설정
- 최대 50명 가입 제한

핵심 포인트:

- 이메일 인증보다 운영 부담이 적은 흐름을 우선 채택
- 사용자별 데이터는 `RLS`로 분리

## Ledger Core

가계부의 기본 흐름은 아래와 같습니다.

1. 자산을 만든다
2. 수입 또는 지출 거래를 입력한다
3. 월별 요약과 목록을 확인한다
4. 필요 시 달력, 검색, 리포트로 다시 본다

지원 기능:

- 수입 / 지출 / 투자 거래 등록
- 월별 요약 카드
- 거래 목록
- 거래 검색
- 달력 기반 확인
- 카테고리 직접 관리

## Assets and Card Settlement

자산은 아래 4가지 타입으로 분리합니다.

- 은행
- 현금
- 카드
- 투자

카드 자산에서는 아래 정보를 함께 관리합니다.

- 카드 결제일
- 결제 통장
- 카드 사용 거래
- 결제예정 금액

이 구조 덕분에 카드 사용일과 실제 출금일을 구분해 현금 흐름을 더 현실적으로 볼 수 있습니다.

## Fixed Expenses and Budget

고정지출과 예산은 월 관리에 직접 연결됩니다.

지원 기능:

- 고정지출 등록
- 활성화 / 비활성화 관리
- 카테고리별 예산 설정
- 월간 사용률 확인
- 미지출 고정비 확인

## Investments

투자 탭에서는 생활비와 분리되지 않은 투자 흐름을 관리합니다.

지원 기능:

- 주식 / 코인 / ETF / 펀드 등록
- 국내/해외 시장 구분
- 원통화(`KRW`, `USD`) 기준 평균매수가/현재가 관리
- 심볼 검색 결과 선택 기반 종목 추가
- 현재가 반영
- 실시간 가격 새로고침
- 펀드 현재가 수동 입력
- 평가금 / 손익 / 수익률 계산

현재 운영상 체크 포인트:

- 실시간 시세 새로고침 Edge Function 검증
- 검색 결과 품질과 종목 선택 UX 확인
- 외부 시세 API 실패 시 메시지 개선

## Operations

일상 사용 외에도 운영 편의 기능을 지원합니다.

- JSON 백업
- JSON 복원
- GitHub 기반 변경 이력 관리
- GitHub Pages 제품 소개 페이지
- README / 명세서 / 세션 로그 문서화
