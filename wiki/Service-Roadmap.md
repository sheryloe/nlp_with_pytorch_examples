# Donggri Ledger Service Roadmap

## 서비스 목표

개인 재무 흐름을 읽고 다음 행동까지 연결해 주는 생활형 가계부 서비스로 발전시킵니다.

## Priority 1

- 카드 결제 예정, 예산 초과, 가격 갱신 실패 알림을 사용자 메시지로 노출합니다.
- 가족 공유 모드와 읽기 전용 뷰를 검토합니다.
- Supabase RLS 테스트와 백업 검증 루틴을 자동화합니다.

## Priority 2

- 기간별 소비 패턴과 예산 편차를 더 쉽게 읽는 대시보드를 추가합니다.
- 사용자가 자주 확인하는 월간 요약과 결제 일정 요약을 강화합니다.
- 운영자가 데이터 이상을 빨리 찾을 수 있게 로그와 진단 화면을 보강합니다.

## Ops

- 배포 절차와 장애 대응 문서를 최신화합니다.
- 외부 API 가격 갱신 실패 시 수동 보정 절차를 문서화합니다.
